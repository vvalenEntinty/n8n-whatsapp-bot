# WhatsApp Bot Automation with n8n

This project is a simple WhatsApp automation bot built using n8n.

## Description

The bot interacts with clients through WhatsApp using a numbered menu system.  
When a user sends a message, the workflow processes the request and stores the client's information automatically in an Excel spreadsheet.

This type of automation helps businesses organize customer data and reduce manual work.

## Technologies Used

- n8n
- Webhooks
- WhatsApp API (Twilio)
- Excel / Google Sheets
- Workflow Automation

## How it Works

1. The user sends a message to the WhatsApp bot.
2. The webhook receives the message in n8n.
3. The workflow processes the request.
4. The bot responds with a numbered menu.
5. The user's response is captured.
6. The information is stored automatically in an Excel spreadsheet.

## Workflow

The workflow is exported as:

workflow.json

You can import it directly into n8n.
